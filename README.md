# Spring Internals

A repository dedicated to understanding **Spring Framework** and **Spring Boot** from the ground up.

Instead of only learning *how to use* Spring, this repository focuses on understanding **how Spring works internally**, accompanied by simple demo projects for every concept.

---

### Repository Convention

Every topic follows the same structure:

- **README.md** → Concept explanation, internal working, common mistakes, and interview questions.
- **code/** → A standalone Spring Boot project demonstrating the concept in practice.
